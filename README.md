#  KFC Landing Page (React)

A responsive KFC-inspired landing page built using React, HTML5, CSS3, and JavaScript.

## Features
- Responsive Navigation Bar
- Smooth Hover Effects & CSS Transitions
- Modern Hero Section
- Interactive Buttons
- Responsive Design
- Clean UI

## Technologies Used
- React
- JavaScript (ES6)
- HTML5
- CSS3
- Vite

## Getting Started

```bash
npm install
npm run dev
```
